• Space or any mouse button: Enable dragging the map while on the canvas.
  • Mouse Drag: Add a line to represent a route segment.
  • Shift + Mouse Drag: Move an existing point if the cursor is near it.
  • Ctrl + Mouse Drag: - Remove a point (and connected lines) if near one.
    - Remove a line if near one.
  • m: Mark the point as a level change point (for stairs, later).
  • p: Toggle the private state of the point.
  • t: Edit the tags of the point.
  • e: Add a room ID to the point.
  • o: Mark the point as outside.
  • c: mark all missing rooms.
  • a: Toggle the accessibility of the line.
  • l: Toggle the locked state of the line.
